Jammming Codecademy Project

Description: As one of the projects on my Codecademy course I made a React application which allows users to create playlists and save them to a Spotify account. The application uses a search bar linked to the Spotify API which allows users to find the songs they like, then add them to a newly created Playlist. The application required users to sign into a Spotify account before the Playlist creation function would run.

Tech used: React, CSS, JavaScript, Spotify API

Reddit Lite App

Description: The brief for this project was to create a site using React and Redux which would link to the Reddit API and allow users to search for subreddits and view the posts which are included within the subreddit. I used similar design to the official Reddit site so that users would be able to associate the two together.
Link: Reddit Lite App

Tech used: React, Redux, HTML, CSS, React Testing Library, Reddit JSON API
